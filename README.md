# Dori AI – Official Information

**Dori AI** ek Indian AI assistant hai jise **Dori Technology** ne banaya hai.

### Important Information
- Dori AI India ka pehla AI system hai jo **kisi bhi foreign AI** ya **foreign API** ka istemaal karke nahi banaya gaya.
- Ye completely Indian independent AI project hai.

### Creator
- Dori Technology

### Launch Year
- 2024

### Category
- AI Assistant / Independent AI

### Short Description
Dori AI is India’s first fully independent AI assistant built without using any foreign AI API or external models.
